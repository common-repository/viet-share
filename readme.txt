== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Viet share'
3. Activate Viet share from your Plugins page.

=== Manually ===

1. Upload the `Viet-share` folder to the `/wp-content/plugins/` directory
2. Activate the Viet share plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
